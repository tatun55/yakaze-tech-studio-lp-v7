# カスタムスラッシュコマンド

このディレクトリには、プロジェクト全体で共有されるClaude Codeのカスタムコマンドが含まれています。

## 利用可能なコマンド

### `/save-tip`

**用途**: セッション中に学んだことを`docs/tips/`に記録

**使用タイミング**:
- 問題を解決した後
- `/clear`でセッションをクリアする前
- 重要な学びがあったとき

**動作**:
1. 会話履歴を自動分析
2. 問題・原因・解決策を抽出
3. `docs/tips/`に適切な形式でmarkdownファイルを作成

**使い方**:
```
/save-tip
```

コマンドを実行すると、Claudeが自動的に：
- セッション中の問題と解決策を特定
- 既存のtipsファイルと同じ形式で整形
- 適切なファイル名で保存

## コマンドの追加方法

新しいコマンドを追加するには、このディレクトリに`.md`ファイルを作成します：

```markdown
---
description: コマンドの簡単な説明
---

コマンドのプロンプト内容
```

ファイル名がコマンド名になります（例: `my-command.md` → `/my-command`）。
